Description:

Made simple, this library allows you to create editable elements on your page. It can be used with any engine (bootstrap, jquery-ui, jquery only) and includes both popup and inline modes. Please try out demo to see how it works. .

How to use:

HTML 
<TagName class="editme" data-edit-"" data-id-"" > Content Here </TagName>
  1. Add class name class="editme"
  2. Add data attribute data-edit="" Leave blank for text input only
  3. Optional for fields id and name use data-id=""

Data Attribute Engine:

Data edit
  • data-edit=""
  • data-edit="select"
  • data-edit="select-only"
  • data-edit="textarea"
  • data-edit="textarea-only"
Data properties (Optional)
  • data-box-width="number"
  • data-box-height="number"
  • data-bg-color="#FFFFFF"
  • data-addClass="classnames"
  • data-note="string"
  • data-placeholder="value"
Data properties select (Optional)
  • data-option-arr="option1 otion2 option3"
  • data-json-url="path/myjson.json"
  • data-json-obj="json object name"
